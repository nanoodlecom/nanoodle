🤖 nanoodle in your coding agent

One command teaches your agent to design and run nanoodle workflows — from your terminal, on your own NanoGPT key.

Built up a folder of saved graphs? Serve them all as tools with the MCP server.

Every graph in the folder becomes a typed tool your agent can call.

claude mcp add nanoodle --env NANOGPT_API_KEY=your-key -- npx -y nanoodle-mcp --graphs ~/noodles

swap your-key for your NanoGPT API key — or sign in here first and it's filled in for you
